<h3>How to use ES6 modules in your browser</h3>
<p>You can include a module in your HTML by adding a script with <mark><code>type="module"</code></mark>:</p>

```html
<script type="module">
    import {helloWorld} from './hello-world.js';
    helloWorld();
</script>
```

```javascript
const helloWorld = () => alert ('Hello World!');
export {
    helloWorld
};
```

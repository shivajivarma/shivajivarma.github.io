How to use ES6 modules in your browser

You can include a module in your HTML by adding a script with type="module":

<script type="module">
    import {helloWorld} from './hello-world.js'; 
    helloWorld();
</script>

const helloWorld = () => alert ('Hello World!');
export {
    helloWorld
};

Computing Greatest Common Divisor using C language

Posted on February 28, 2014 by Shivaji Varma

The greatest common divisor (GCD) of two or more integers (at least one of which is not zero), is the largest positive integer that divides the numbers without a remainder.

For example, the GCD of 8 and 12 is 4.

Code

#include<stdio.h>

  int gcd(int a,int b){  
	int c=b%a;
	
	if(c == 0) return a;
	else return gcd(c,a);
  }

int main(){
  
	int x,y;
	
	printf("GCD of two numbers\nEnter x and y:: ");
	scanf("%d%d",&x,&y);
	
	if(x==0 && y==0) printf("Error: Atleast one of the interger must not be zero.");
	else{
	  if(x < y) printf("\n GCD:: %d",gcd(x,y));
	  else
		printf("\n GCD:: %d",gcd(y,x));
	}
	return 0;
}

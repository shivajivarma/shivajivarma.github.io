How To Host A Website On Github

Posted on February 4, 2016 by Shivaji Varma

Steps

  • Create an account on github
  • Create a repo with name 'yourusername.github.io'
  • Put the website files in the repo
  • Thats it.. your website is ready and running at http://yourusername.github.io
